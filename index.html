Richard's Academy

Holographic Study Interface v2.0

WHAT IS THIS?

Richard's Academy is a futuristic-themed personal study resource hub designed to organize academic modules, lecture slides, and subject files across all year levels.

FEATURES

  • ◈ Organized by Year Level and Semester
  • ◈ In-page PDF and file viewer
  • ◈ Holographic UI with cyberpunk aesthetics
  • ◈ Responsive for desktop and mobile
  • ◈ XP progression system
  • ◈ Ambient cyberpunk music player
